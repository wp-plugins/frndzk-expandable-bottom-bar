Frndzk Expandable Bottom Bar plugin adds an expandable botttom bar in your website.

== Description ==

Frndzk Expandable Bottom Bar plugin adds a dashing expandable botttom bar in your website. you can set the bottom bar contents from widgets area. The bar expands when you moves your mouse on it. You can set the footer text and other settings from plugin's settings area. [demo](http://bitto.us/wp/) plugin developed by [bitto kazi](http://bitto.us)

== Installation ==

1. Upload 'frndzk-expandable-bottom-bar.zip' to the '/wp-content/plugins/' directory and extract the zip file
2. Activate the plugin through the 'Plugins' menu in WordPress
